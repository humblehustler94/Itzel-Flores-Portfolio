## ğŸš€ Quick Start

This project is a static website, so no complex build process or server setup is required to get started locally.

### Prerequisites
-   A modern web browser (e.g., Chrome, Firefox, Safari, Edge) to view the website.
-   A code editor (e.g., VS Code) for development.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/humblehustler94/Itzel-Flores-Portfolio.git
    cd Itzel-Flores-Portfolio
    ```

2.  **Open in your browser**
    Simply open the `index.html` file directly in your web browser, or use a local development server for convenience (e.g., VS Code's Live Server extension).
    
    If using a local server, navigate to `http://localhost:[port]` (default for Live Server is often 5500).
